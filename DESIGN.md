# Design DNA — Stackwise

## Visual Personality: technical-precise
## Layout: enterprise
## Typography: modern-geometric

### Color Palette
- Primary (Deep Navy):   #0F1F3D  → oklch(0.17 0.06 250)
- Accent (Teal):         #2EC4B6  → oklch(0.72 0.12 192)
- Surface (Off-White):   #F4F6FA  → oklch(0.97 0.008 250)
- Text (Light on dark):  #E8EDF5  → oklch(0.92 0.012 250)
- Text (Dark on light):  #1A2B4A  → oklch(0.23 0.05 250)

### Font Pairing
- Display: Space Grotesk — weights 400/500/600/700 (sharp, technical, modern)
- Body: DM Sans — weights 300/400/500 (readable, clean, professional)

### Unique Design Choices
1. **Data-grid hero overlay** — subtle CSS dot-grid + faint horizontal rule pattern over deep navy hero, evoking Bloomberg Terminal / Palantir intelligence aesthetics. Layered with asymmetric teal glow ellipses offset top-right.
2. **Stats bar with large tabular figures** — full-width bar below the hero using alternating teal numerals + slate labels, visually breaking the page into distinct authority zones.
3. **Problem/Solution split layout** — left side lists "What generic directories miss," right side contrasts "What Stackwise delivers" with teal-accented cards — a direct, visual competitive argument.
4. **Trust/compliance badges row** — inline SVG icons for SOC 2, ISO 27001, GDPR, FedRAMP placed prominently, reinforcing enterprise-grade credibility early.
5. **Dark navy hero + footer, neutral content mid-sections** — creates a "frame" effect; interior sections use cool off-white for readability, mirroring enterprise dashboard software aesthetics.
